Aide Badges visiteurs
Ajouter un badge
  • Cliquez sur Scanner et ajouter puis approchez le badge du lecteur.
  • Optionnel : donnez un nom au badge (affiché dans la liste).
  • Les badges visiteurs n’appartiennent à personne par défaut.
Affecter un badge
  • Depuis l’accueil: scannez un badge non affecté → popup d’affectation.
  • Vous pouvez créer un utilisateur sur le moment (sans carte personnelle).
  • L’affectation est autorisée aux non-admins. Cette page reste admin-only.
  • Important : les utilisateurs blacklistés ne peuvent pas être affectés à un badge visiteur.
Désaffecter / supprimer
  • Bouton Désaffecter pour libérer le badge.
  • Bouton Supprimer pour retirer le badge de l’inventaire.
  • Un reset automatique désaffecte tous les badges chaque jour à 00:00.
Bonnes pratiques
  • Pour rompre l’attache d’une carte personnelle à un utilisateur: utilisez Détacher la carte dans Utilisateurs.
  • Donnez un nom aux badges visiteurs pour les identifier plus facilement.
